# Calendar Click V1.1 Enhancements

## Problem Frame

Calendar Click V1 delivers the core one-click copy experience. Six friction points remain that reduce daily usability: mouse-only activation, robotic time formatting, a hidden setting, no pre-copy verification, no cross-timezone support, and single-format output. These enhancements collectively move the app from "useful" to "indispensable" for users who schedule multiple times per day.

## Requirements

### R1. Global Keyboard Shortcut

- R1.1. A configurable system-wide hotkey triggers the default copy action (same as left-click)
- R1.2. Default shortcut: Ctrl+Shift+C
- R1.3. Shortcut picker in Settings (standard macOS key-recording UI)
- R1.4. Same checkmark/X-mark feedback after shortcut triggers
- R1.5. Shortcut can be disabled (cleared) in Settings

### R2. Smart Slot Rounding

- R2.1. Free-slot start times round UP to the nearest increment (10:47am becomes 11:00am)
- R2.2. Free-slot end times round DOWN to the nearest increment (2:08pm becomes 2:00pm)
- R2.3. Rounding increment is configurable: 5, 10, 15, or 30 minutes
- R2.4. Default rounding increment: 30 minutes
- R2.5. Rounding picker in Settings (dropdown)
- R2.6. After rounding, slots shorter than the minimum slot duration are filtered out
- R2.7. A "No rounding" option (off) is available for users who want exact times

### R3. Expose Minimum Slot Duration Setting

- R3.1. A picker in Settings for minimum slot duration
- R3.2. Options: 15 min, 30 min, 45 min, 1 hour
- R3.3. Default: 30 minutes (matches current behavior)
- R3.4. The backend key and clamped getter already exist — only the UI is missing

### R4. Preview Popover

- R4.1. Option+click the menu bar icon opens a popover showing the formatted availability text
- R4.2. Popover anchored under the menu bar icon (same positioning as Settings window)
- R4.3. Popover includes a "Copy" button that copies and dismisses
- R4.4. Popover includes a timezone picker dropdown for recipient timezone conversion (see R5)
- R4.5. Popover dismisses on click-outside or Escape
- R4.6. Left-click behavior unchanged — still copies instantly without preview
- R4.7. Popover stays on top (floating level, same as Settings)

### R5. Recipient Timezone Conversion

- R5.1. The preview popover (R4) includes a timezone picker dropdown
- R5.2. Default selection: "My timezone" (no conversion)
- R5.3. When a different timezone is selected, all times in the preview convert to that timezone
- R5.4. The timezone label on the output changes to the recipient's timezone (e.g., "(ET, GMT-5)")
- R5.5. Recent/favorite timezones remembered in Settings (last three used)
- R5.6. Timezone search — user can type a city name or abbreviation to filter
- R5.7. Left-click (no preview) always uses the sender's local timezone — conversion only through the popover

### R6. Format Templates

- R6.1. Two output formats: Plain Text (default) and Markdown
- R6.2. Default format selectable in Settings
- R6.3. Markdown format uses bullet list: `- **Mon Mar 25:** 9-10:30am, 2-3pm`
- R6.4. Format selector available in the preview popover (R4)
- R6.5. Left-click uses the default format from Settings
- R6.6. Right-click menu items also use the default format

## Success Criteria

- Users can copy availability without touching the mouse (keyboard shortcut)
- Pasted times look human-authored, not machine-precise (rounding)
- Users can verify output before it reaches a recipient (preview)
- Cross-timezone scheduling requires no mental math (conversion)
- Output is native to the destination — plain text for SMS, Markdown for Slack (templates)

## Scope Boundaries

- No rich text / HTML format in this version (defer to V1.2 if needed)
- No drag-and-drop from the icon
- No auto-paste into active text fields
- No contextual scheduling detection
- No availability diffing (deferred to later ideation)
- Keyboard shortcut is for the default copy only — no shortcut for preview or right-click menu

## Key Decisions

- **Ctrl+Shift+C as default hotkey**: C for Calendar/Copy, avoids common conflicts
- **Rounding is configurable (5/10/15/30 min), default 30**: Users who schedule 15-min calls can reduce it; executives who only do hour blocks can increase it
- **Option+click for preview**: Preserves the instant left-click flow while adding a safety net
- **Timezone picker lives in the popover, not the right-click menu**: Combines naturally with preview — see the output, adjust timezone, then copy
- **Plain text + Markdown only**: Two formats keeps it simple. HTML deferred.
- **Left-click is never affected by timezone/format selections in the popover**: The instant flow always uses defaults from Settings. Popover is the "customized copy" path.
